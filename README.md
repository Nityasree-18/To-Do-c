#  Task Manager (To-Do List App)

A simple and efficient **Task Manager web application** built using **HTML, CSS, and JavaScript**.
This app allows users to manage daily tasks with features like adding, completing, and deleting tasks — all without using any database.

---

##  Features

*  Add new tasks
*  Mark tasks as completed
*  Delete tasks
*  Data stored using **Local Storage** (no database required)
*  Clean and responsive UI

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript
* Browser Local Storage

---

##  Project Structure

```
todo-app/
│── index.html
│── style.css
│── function.js
```

---

##  How to Run the Project

1. Download or clone this repository
2. Make sure all files are in the same folder
3. Open `to_do.html` in any web browser
